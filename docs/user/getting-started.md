# Getting Started

Use this page to get from a fresh launch to a working collection.

## Launch the App

When Second Brain opens, the sidebar is on the left and the main content area is on the right.

- Click `Dashboard` in the sidebar at any time to return to the collection overview.
- Click `Settings` in the sidebar to open global backup and archive tools.

## Create Your First Collection

1. In the sidebar, click `New Collection`.
2. Enter a name in the `Create New Collection` dialog.
3. Click `Create Collection`.

The new collection appears in the `Collections` section of the sidebar.

## Open the Collection

1. Click the collection name in the sidebar.
2. The collection expands and shows its views.
3. Click the locked source view to open the collection.

Every collection starts with a default source view. This view is the base schema and field order for the collection.

## Add Fields Before Adding Data

If the collection has no fields yet, the grid shows a `No Fields Yet` empty state.

1. Click `Add Fields` in the grid, or switch to the `Fields` tab in the top bar.
2. Add the fields you want to store.
3. Return to `Data` when the schema is ready.

## Add Your First Item

Once fields exist, the grid shows an add row at the bottom with a `+` cell.

1. Click the `+` row.
2. A new item is created with the field defaults you defined.
3. The first field immediately enters edit mode so you can start typing.

For a fuller explanation of field setup and item editing, continue to [Fields](./collections/fields) and [Items](./collections/items).
